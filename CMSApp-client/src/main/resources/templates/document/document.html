HTML Basics

HTML, the Hypertext Markup Language, is the standard markup language for creating web pages and web applications. With Cascading Style Sheets (CSS) and JavaScript, it forms a triad of cornerstone technologies for the World Wide Web.

What is HTML?

HTML is a markup language used to structure content for display in a web browser. It uses a series of elements, represented by tags enclosed in angle brackets (< >), to mark up the content of a web page.

HTML Document Structure

A basic HTML document has the following structure:

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>
  </body>
</html>
